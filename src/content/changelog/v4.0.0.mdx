## MEWO cursor

If you're on desktop, try hovering over the quote on the homepage. You should see MEWO replace your cursor as long as it remains on the quote.

Honestly, after adding the previous text animation, I wasn't too sure what else to do with it. I figured that since I removed the quote author, I might as well add it back some other way.

Unfortunately MEWO has black fur, so it's a bit hard to see. More reason to add a sitewide light mode.

## Other changes in v4.0.0

- adjust spacing for everything
- create more specific layouts on top of a base layout (e.g. include header and footer already)
- Revert `<MewoQuote />` back from being a web component
- Rewrite bio on homepage. Add more about my school clubs :\)
- Add more stuff to OG metadata, tell browser my site prefers dark
- Add `subtitle` as a Tailwind utility. Sometimes standard text feels a bit too bright for what I'm trying to convey, so I set its opacity to 90% and slightly color it.
- Delete `<NotFound />` because I realized you could directly import entire Astro pages as a component! So I can just import and use my site-wide 404 page in everything else.
- Display date on OG metadata for each post (instead of tags, which have been destroyed)

Really utilize Tailwind `@layer` directives more. I've moved a lot of unruly in-component directives to `base.css`. For example, I inlined the CSS for the MEWO cursor, which now has an absolute path, meaning in theory I could style anything with it.

Instead of being inlined like this on the HTML element

```html
<p class="cursor-[url('../assets/mewo.webp')_32_32,_not-allowed]">Incredible quote.</p>
```

It's now easier to read like this:

```css
@layer utilities {
  .mewo-quote {
    cursor:
      url("/src/assets/mewo.webp") 32 32,
      not-allowed;
  }
}
```

Another use case was with Berkeley Mono: its weight range spans from 100 to 150. Since regular weight is usually 400, I always had to specify `font-[100]` right after using `font-mono`. I also prefer my monospaced fonts slightly smaller than normal, so it was usually also followed by a `font-[14.5px]`. Now all of that's gone.
